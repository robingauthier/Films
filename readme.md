

# Comment trouver les films
Il faut chercher sur yandex.com et tu trouveras des liens ok.ru ou yandex.com/video
Les russes connaissent tres bien les vieux films francais. 


Je ne mentionne pas les films trop connus (les gendarmes, le corniaud, le samourai,fantomas,la boum, les miserables, le compte de monte Cristo, ... ).  Le but est de parler de ces films oublies qui sont tres bien. 

## Acteurs
- Patrick Deware : immense acteur a mon avis. Il s'est suicide jeune. "Serie noire" est un chef d'oeuvre. Les valseuses aussi. 
- Michel blanc : c'est aussi un bon realisateur.   Marche a l'ombre se regarde bien.
- Patrice Leconte : Tres bon realisateur qui n'a pas fait que  les bronzes. Viens chez moi j'habite chez une copine : c'est marrant. La musique de Renaud est connue. Les grands ducs c'est marrant aussi. "Ridicule" aussi est tres bien.
- Michel Serrault : Immense acteur. Le film "Garde a vue" est super. 
- Annie Girardot : Jeune elle avait les cheveux long ca lui allait mieux. Immense actrice. 
- Jean Pierre Marielle : immense acteur. Un de mes preferes. 
- Philippe Noiret : immense acteur.
- Alain Delon  : j'ai ete decu par "La piscine". "Le samourai" super,
- Belmondo : faut voir "Leon morin pretre" pour voir comment il a commence.
- Fernandel : j'aime bien mais il a tres mal vieilli. Don Camillo c'est bien mais ca fait vieux aujourd'hui. "La vache et le prisonnier" tres bien. 
- Bourvil : c'est l'acteur apprecie par les gens des campagnes. "La cuisine au beurre" se regarde bien et est moins connu que le corniaud.

## Realisateurs
- Michel Audiard : realisateur mais surtout dialogiste. Le genie du dialogue francais. Gabin est tres bon pour interpreter les dialogues de Audiard. 
- Marco Ferreri : Le mec a un grain de folie. _Liza_ (1972) et _La Grande Bouffe_ (1973) sont bien. 
- Andre Hunebelle : a realise les vieux OSS 117 qui se regardent bien. Ils ont moins d'humour que les versions Dujardin mais c'est pas mal quand meme.
- Sacha Guitry : pas un grand fan, trop parisien a mon gout. Et puis je n'aime pas le theatre, or les films de Sacha Guitry font tres pieces de theatre.
- Henri Decoin : tres grand realisateurs. Beaucoup de choses a voir chez lui. 
- Rene Clement : Tres bien. Jeux interdit est bien, beaucoup de films sur la guerre et resistance.
- Gilles Grangier :
- Marcel Pagnol : faut tout regarder. Faut aussi regarder les adaptations de Yves Robert ( La gloire de mon pere, le chateau de ma mere).

## Liste de films sans ordre a absolument voir


- La grande bouffe (https://fr.wikipedia.org/wiki/La_Grande_Bouffe). Casting phenomenal, mais film qui derange, choque. D'ailleurs comme tous les films de Marco Ferreri. Pour autant c'est un grand film. Il faut le voir plusieurs fois pour l'apprecier.
- Serie noire (https://fr.wikipedia.org/wiki/S%C3%A9rie_noire_(film,_1979)) 
- Calmos ! : film scandaleux aujourd'hui. Il faut le voir je pense. la fin est tres decevante par contre a mon avis. Marielle est tres marrant a mon avis.
- Plein Soleil : tres bien. Un grand classique je dirais.
- L'etudiante : extraordinaire avec Sophie marceau (https://ok.ru/video/272647064127) 
- Les films de marcel pagnol : ( la fille du puisatier, Marius, Fanny). Les versions de Daniel Auteuil sont mieux que les anciennes version a mon avis. 
- Le sucre (https://fr.wikipedia.org/wiki/Le_Sucre) bien sur la bourse.
- Le trio infernal (https://fr.wikipedia.org/wiki/Le_Trio_infernal) excellent
- Les aventuriers (https://fr.wikipedia.org/wiki/Les_Aventuriers_(film,_1967))
