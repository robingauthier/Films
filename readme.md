

# Comment trouver les films
Il faut chercher sur yandex.com et tu trouveras des liens ok.ru ou yandex.com/video
Les russes connaissent tres bien les vieux films francais. 


Je ne mentionne pas les films trop connus (les gendarmes, le corniaud, le samourai,fantomas,la boum, les miserables, le compte de monte Cristo, l'aventure c'est l'aventure, le guignolo, flic ou voyou, les bronzes, le pere noel est une ordure, les sous doues , les mousquetaires, cyrano de bergerac... ).
